# 更新日志

## [1.0.0] - 2026-01-09

### 新增功能

- 🎨 全新的二次元风格界面设计
- 🌓 支持深色/浅色主题切换
- 🎵 完整的歌单管理系统
- 🔍 强大的筛选和搜索功能
- 📋 一键复制歌曲名
- 🎛️ 管理后台系统
- 📦 JSON 格式导入导出
- 🔧 XLSX 转 JSON 工具
- 🐳 Docker 容器化部署
- 🚀 GitHub Actions CI/CD
- 📄 完整的文档和示例

### 技术栈

- 后端：Node.js + Express + SQLite
- 前端：React 18 + MUI v7 + Vite
- 部署：Docker + Docker Compose
- CI/CD：GitHub Actions

### 特色功能

- 液态玻璃效果（Glassmorphism）
- 渐变色主题系统
- 防抖搜索
- 分页浏览
- 文件上传
- 数据备份恢复
